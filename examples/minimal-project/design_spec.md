# Design Spec

## Project Summary
- 项目名称：quarterly-growth-review
- 一句话目标：面向管理层做季度增长复盘，结论先行，突出增长来源、关键风险与下季度动作。
- 主要受众：管理层

## Canvas Format
- Format: ppt169
- ViewBox: 0 0 1280 720
- 目标页数：8

## Visual Style
- 风格关键词：商务清晰、数据优先、结论先行
- 推荐主题：consulting
- 色彩建议：深蓝、灰白、少量强调色
- Typography：标题强对比，正文尽量短句

## Image Strategy
- 图像策略：图标 + 图形化表达
- 模板策略：free-design

## Content Outline
1. 封面
2. 执行摘要
3. 增长驱动因素
4. 渠道与效率表现
5. 问题与风险
6. 关键动作建议
7. 下季度计划
8. 结束页
