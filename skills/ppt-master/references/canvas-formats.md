# Canvas Formats

下面是从 `ppt-master` 项目整理出的常用画布格式，供 WorkBuddy 技能在确认输出规格时直接引用。

## 演示文稿

### PPT 16:9
- viewBox: `0 0 1280 720`
- 比例: 16:9
- 场景: 商务汇报、项目路演、现代会议屏幕
- 默认推荐: 是

### PPT 4:3
- viewBox: `0 0 1024 768`
- 比例: 4:3
- 场景: 老投影仪、教室展示、传统会议室

## 社交媒体与内容卡片

### 小红书图文
- viewBox: `0 0 1242 1660`
- 比例: 3:4
- 场景: 知识卡片、图文种草、封面页

### 朋友圈 / Instagram 方图
- viewBox: `0 0 1080 1080`
- 比例: 1:1
- 场景: 方形海报、活动宣传、品牌图

### Story / TikTok 竖版
- viewBox: `0 0 1080 1920`
- 比例: 9:16
- 场景: 短视频封面、竖版故事、信息流

## 营销物料

### 微信文章头图
- viewBox: `0 0 900 383`
- 比例: 约 2.35:1
- 场景: 公众号头图、专题文章封面

### 横版 Banner
- viewBox: `0 0 1920 1080`
- 比例: 16:9
- 场景: 网页 Banner、横屏展示、大屏轮播

### 竖版海报
- viewBox: `0 0 1080 1920`
- 比例: 9:16
- 场景: 手机海报、电梯屏、会场竖屏物料

### A4 Print
- viewBox: `0 0 1240 1754`
- 比例: 1:√2
- 场景: 打印海报、单页传单、纸质资料

## 选择建议
- 不确定时：优先用 PPT 16:9
- 老旧投影场景：用 PPT 4:3
- 社媒传播：按平台比例走，不要硬塞进 PPT 比例
- 既要演示又要传播：建议先做 PPT 主版，再衍生社媒版

## 给代理的提醒
在开始生成之前，务必明确：
1. 用户要的是演示稿还是营销物料
2. 最终展示设备是什么
3. 是否需要多端复用
4. 是否需要同时产出多种比例版本
